📜 Gat0B0t Commands

🔨 Moderation
  • /swarn — Silently warn a member via DM.
  • /skick — Silently kick a member.
  • /sban — Silently ban a member.
🎲 Fun
  • /dice — Roll a single dice (1–6).
  • /roll — Roll 3 random numbers (1–7).
  • /rps — Play rock-paper-scissors with the bot.
  • /tictactoe — Play Tic-Tac-Toe with buttons.
  • /roast — SFW roast a user.
  • /say — Make the bot say something.
  • /airhorn — Play an airhorn in a voice channel.
  • /vineboom — Play the iconic vine boom sound in VC.
  • /dadjoke — Receive a random dad joke.
  • /badtranslate — Translate text through multiple languages for chaos.
🛠️ Utility
  • /membercount — Get number of server members.
🖼️ Damn Bird Features
  • /uploaddamn — Upload one or more images from URL(s).
  • /randomdamn — Send a random image from uploads.
  • /damnhelp — Sends DM instructions on using the damn features.
📈 Leveling System
  • /enableleveling — Enable leveling system.
  • /rank — Check your level and XP.
  • /lvlcooldown — Set XP gain cooldown per server.
  • /dailyxp — Claim daily XP (level 5+).
  • /leaderboard — Show server leaderboard.
🎵 Music
  • /playmusic — Play music from a YouTube, Soundcloud, Newgrounds (rare, ik) or Spotify link.
  • /stopmusic — Stop the currently playing music and clear the queue.
